OpenPGP

Pretty Good Privacy (PGP) is a data encryption standard created by Phil Zimmermann in 1991. It uses a combination of symmetric-key cryptography and public-key cryptography. it assures both confidentiality and authenticity of the information.

OpenPGP.js is an open-sourced implementation of PGP written in JavaScript. It offers three functions, openpgp.generateKeyPair() to generate a public-private key pair, openpgp.encryptMessage() to encrypt messages and openpgp.decryptMessage() to decrypt messages.

Potential concerns

- SSL is still a requirement, even with this kind of exchange. - The private key should never be shared with anyone and shouldn't be put on a server. - Encryption should be applied with care as the operation is not without a cost.

A note on window.crypto

window.crypto is the native javascript cryptography library. As of right now, the most supported operation is window.crypto.getRandomValues(). The salting demo of this app uses getRandomValues() to generate a random salt.

There is a Web Cryptography API standard in the works which can be accessed through the window.crypto.subtle object. This includes a lot of the same functionality as OpenPGP.js and some more. OpenPGP.js is set up to use the Web Cryptography API when it is available and fall back to the OpenPGP.js implementation when it is not.
